Hexo更换主题butterfly

发表于2024-10-26|更新于2024-10-26|Hexo
|总字数:174|阅读时长:1分钟|浏览量:

前言

Butterfly 是一个基于 Bootstrap 开发的响应式主题，适合 Hexo 博客。本文将介绍如何更换 Hexo 博客主题为 butterfly。

准备工作

首先，你需要安装 Hexo。如果你还没有安装，请参考 Hexo 官方文档进行安装。

npm install -g hexo-cli

更换主题

下载主题

你可以通过以下命令下载 butterfly 主题：

git clone https://github.com/jerryc127/hexo-theme-butterfly.git themes/butterfly

修改配置文件

打开 Hexo 配置文件_config.yml，找到theme字段，将其修改为butterfly。

theme: butterfly

重启 Hexo

hexo clean
hexo g
hexo s

其余相关配置,请参考 butterfly 主题的 README.md 文档。
