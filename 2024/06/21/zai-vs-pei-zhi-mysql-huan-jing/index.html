C++ 加 MySQL

打开cmd
输入 mysql
输入 mysql -u root -p

打开VisualStudio
VC/C目录
项目属性:点击c++->常规->附加包含目录->"选择C:\Program Files\MySQL\MySQL Server 8.0\include"

包含库文件->链接->常规->附加库目录->"选择C:\Program Files\MySQL\MySQL Server 8.0\lib"
写入库文件->链接器->输入->附加依赖项->libmysql.lib
libmysql.lib复制到c盘windows下system32文件夹下
